# one-key-core

A _garden_ project

---

One Key Core is a complete module for issuing and limiting API keys using Redis.

It is part of a larger one-key ecosystem inculding a deployable Hono API.

This project powers the underlying system and should only be used directly if the larger one-key project doesn't suit your needs.

A good use case for core, for instance, would be if you would like to add API keys to an existing Express API.

If something like that sounds useful to you, check out our [Getting Started](./guides/getting-started.md)
